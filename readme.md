# О проекте

Тестовое задание для компании Only. Задача заключается в создании блока по макету с использованием анимаций.

## Запуск проекта

- npm run dev: запуск проекта для локальной разработки
- npm run build: сборка проекта

## Особенности проекта

- Компоненты структурированы в виде паттерна atomic design. Готовый блок находится по адресу `components/templates/Block`
- Для создания слайдов использован swiper
- Анимация реализована с помощью gsap
- Проект собран с помощью webpack
